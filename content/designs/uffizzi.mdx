## REST API 

The Uffizzi Controller API is a smart proxy service that handles requests from the primary Uffizzi API to the Kubernetes API. It is designed to provide developers with a secure and reliable way to interact with their environments and applications.

## Command-line (CLI)

The Uffizzi command-line iterface (CLI) allows developers to interact with their environments and applications from a terminal. The CLI is designed to provide developers with a seamless experience for creating and connecting to their environments. It also faciliatates resource management and debugging.

The CLI enables CRUD operations on various Uffizzi resources such as accounts, projects, configs, compose environments, clusters, registries, and more.

[Explore this design →](https://docs.uffizzi.com/install/cli)
